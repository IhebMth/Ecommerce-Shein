A new loadProducts() function fetches from the API and converts the flat field names to the nested format your frontend expects. Nothing in the HTML structure changes at all.
📌 API returns flat fields. Your frontend expects nested objects. The conversion is critical:
API: name_en, name_ar, desc_en, desc_ar, old_price
Frontend needs: name:{en,ar}, desc:{en,ar}, oldPrice
Colors: color.name_en/ar → color.name:{en,ar} and images:[{url}] → images:[url string]
